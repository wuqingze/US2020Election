### Introduction
----------------
今天是美国大选日，这次选举对美国的民主自由和将来的中美关系至关重要。
这次大选可能是百年来重要的一次美国总统选举，将关乎未来10年美国和世界
的历史走向。因此我对大选的投票分布非常感兴趣。这个项目是为了得到选举
结果票数可能的分布情况，并做出我关于选举结果的预测。

我们知道美国采用的是选举人团制度.选举人团（英语：Electoral College）
是美国总统选举的方式，是一种间接选举，旨在选出美国总统和副总统。
根据《美国宪法》，美国各州公民先选出该州的选举人，再由选举人代表该州投票；
不采用普选制度的原因，主要是由于美国是联邦制国家，
并考虑到各州的特定地理及历史条件，制宪元老决定采取选举人团制度，
保障各州权益，所以美国没有公民直选的总统。

现时全国共设有538个选举人团席次，即每个州份各两个、另每个国会选区各一个，
以及华盛顿哥伦比亚特区三个。也就是各种拥有一定比例的选举票数美国一共有50
个州一个华盛顿特区,那么如果我们追求选举结果的随机分布，那么我们将会得到2**51
=2251799813685248种分布情况，这个结果是平反和计算复杂的，
因为现实世界中美国存在红州、蓝州和摇摆州的情况，我们已经知道哪些州会归于民主党，
那些州的选举人票会归于共和党，不确定是摇摆州的选举结果。因此我们可以排列出所
有摇摆州的归属结果，从而减轻计算的复杂度。2020年美国一共有13个摇摆州，
因此我们可以得到2**13=8192种情况。

**我预测拜登vs川普的选举人票是350:190左右，拜登超千万张直选票。**
